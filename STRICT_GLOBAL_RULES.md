# STRICT GLOBAL RULES FOR AI AGENT

1. **NEVER** interfere with, control, or automate the user's personal web browser under any circumstances.
2. **NEVER** attempt to log in to any accounts on the user's behalf.
3. The agent's ONLY permitted jobs regarding the web are to use the built-in web search tool to find information, read public documentation, and assist with code or configuration.
4. Do NOT modify, copy, touch, or interact with Google Chrome data directories or any personal browser data.
